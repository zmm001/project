E都市于2005年圣诞节期间开创性地推出了三维地图线上挖宝活动，引发了网民的广泛关注和积极参与，2005年首届圣诞挖宝活动参与人数超十万，1000人挖到了心仪的奖品。2006年和2007年，E都市陆续开展了第二届和第三届挖宝活动，参与人数呈几何态势迅速增长，近八十万人参与了挖宝活动。

　　随着E都市知名度的提升，挖宝活动的影响力也在迅速增强。2008圣诞前期，E都市通过《都市快报》发送了10万份插页广告对第四届挖宝活动进行了大范围宣传，同时借助杭州本地26家知名网站同步开展挖宝活动的线上推广，实现了第四届挖宝活动前的充分预热。圣诞期间，E都市第四届圣诞挖宝活动如期举行，活动场面火爆，盛况空前，参与人数首次突破百万，同时在线人数最高达到二十万，通过活动共发放奖品3000件，用户好评如潮。2009年预计参与人数可达120万，发放奖品4000件。

　　参与商家品牌在本次挖宝活动中通过首页广告、页卡广告、泡泡广告等多种渠道进行了高频率曝光，展示品牌形象的同时加深了品牌美誉度，为后期拓展客户群体创造了机会。同时，E都市还通过多种附赠的优质营销套餐对商家的品牌及产品进行持续性的推广宣传，实现广告效果的深化和巩固，为商家带去巨额回报。
